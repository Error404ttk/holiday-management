# คู่มือผู้ใช้

คู่มือนี้สำหรับผู้ใช้งานทั่วไปของระบบจัดการวันหยุด HOSxP

## เข้าสู่ระบบ

1. เปิด URL ของระบบ
2. กรอกชื่อผู้ใช้งานและรหัสผ่าน
3. กดปุ่มเข้าสู่ระบบ
4. ถ้ารหัสผ่านไม่ถูกต้อง ระบบจะแสดงข้อความแจ้งเตือน

## ดูรายการวันหยุด

1. เลือกเมนู “รายการวันหยุด”
2. ระบบจะแสดงรายการวันหยุดจากตาราง `holiday`
3. ใช้ช่องค้นหาเพื่อค้นหาชื่อวันหยุด
4. กดไอคอนรูปตาเพื่อดูรายละเอียดรายการ

## เลือกปี พ.ศ.

1. ที่หน้ารายการวันหยุด เลือกช่อง “ปี พ.ศ.”
2. ระบบจะแปลงปี พ.ศ. เป็น ค.ศ. เพื่อค้นหาข้อมูลจากฐานข้อมูล
3. วันที่บนหน้าจอจะแสดงเป็นรูปแบบไทย พ.ศ.

## เลือกปีปฏิทิน / ปีงบประมาณ

- ปีปฏิทิน: แสดงช่วงวันที่ 1 มกราคม ถึง 31 ธันวาคม
- ปีงบประมาณ: แสดงช่วงวันที่ 1 ตุลาคมของปีก่อนหน้า ถึง 30 กันยายนของปีที่เลือก

ตัวอย่างปีงบประมาณ 2569:

- เริ่ม: 01/10/2568
- สิ้นสุด: 30/09/2569

## ตรวจสอบวันหยุด

1. เลือกเมนู “ตรวจสอบวันหยุด”
2. เลือกวันที่
3. กดปุ่ม “ตรวจสอบ”
4. ระบบจะแสดงผลว่าเป็นวันหยุด ไม่ใช่วันหยุด หรือพบข้อมูลแต่ถูกปิดใช้งาน

## ความหมายของสถานะ

- ใช้งาน: รายการวันหยุดยังถูกใช้งาน
- ปิดใช้งาน: รายการยังอยู่ในฐานข้อมูล แต่ไม่ใช้เป็นวันหยุดในระบบ
- ไม่รองรับ: schema ของตาราง `holiday` ไม่มี field สำหรับสถานะ active/status

## ออกจากระบบ

กดปุ่มออกจากระบบที่แถบเมนู ระบบจะล้าง token และกลับไปหน้า login
